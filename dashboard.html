INTEGRATION GUIDE
// Three ways to plug AI Stack Doctor into your ecosystem
01
Command Line
The fastest way to get started. Run audits interactively, export to PDF or TXT, and browse history — all from your terminal.
# Install dependencies pip3 install anthropic ddgs rich flask # Set your API key export ANTHROPIC_API_KEY="sk-..." # Run interactive audit python3 ai_stack_health_agent_v3.py # Browse history python3 ai_stack_health_agent_v3.py --history # Start dashboard python3 dashboard_server.py
02
REST API
Expose the agent as a local REST endpoint. Integrate with your existing tooling, CI/CD pipelines, or internal dashboards.
# Start the API server python3 ai_stack_health_agent_v3.py --api # Run an audit via POST curl -X POST http://localhost:8080/audit \ -H "Content-Type: application/json" \ -d '{"company":"Stripe","mode":"competitor"}' # Get history curl http://localhost:8080/history # List top-tier companies curl http://localhost:8080/companies
03
Python Module
Import directly into your Python codebase. Use the agent programmatically, export PDFs, or chain with your own data pipelines.
from ai_stack_health_agent_v3 import ( run_agent, save_to_history, get_last_report, export_report_to_pdf ) from pdf_export import export_report_to_pdf # Run a programmatic audit report = run_agent("Stripe", "competitor") # Export to dark-themed PDF path = export_report_to_pdf(report, "Stripe") print(f"PDF saved: {path}")
ECOSYSTEM INTEGRATION
// Embedding AI Stack Doctor into your technology workflow
01
Quarterly Audit Cadence
Schedule recurring audits using cron or your existing orchestration tool (Airflow, Prefect, GitHub Actions). The SQLite history DB automatically tracks deltas so you can see improvement or regression between runs. Recommended: quarterly for competitors, monthly for your own stack.
02
CI/CD Integration
Use the REST API endpoint in your deployment pipeline. Trigger a self-audit after major infrastructure changes to track how architectural decisions affect your AI stack health score over time. Set score thresholds to gate releases.
03
Executive Reporting
The PDF export produces a board-ready dark-themed report with score rings, peer benchmarking tables, and executive summaries. Embed into your quarterly business review cycle or investor update. Compatible with Google Slides, PowerPoint, and Notion.
04
Competitive Intelligence Workflow
Build a competitor monitoring dashboard by scheduling weekly audits of key competitors. The delta tracking system will surface meaningful shifts in their AI stack — new tool adoptions, governance changes, or platform migrations — before they appear in press releases.
05
Open Source & Extension
The entire codebase is open source (MIT license). Add new companies to the intelligence layer by extending COMPANY_INTEL. Add new scoring categories by updating the system prompt and tool definitions. Contribute back via the GitHub repo at github.com/dwnjuguna/AI-Stack-Doctor.
